B1: Tải yarn để tạo project nhanh hơn:
Vào cmd gõ : choco install yarn
------------------- Bỏ qua nếu đã tải yarn -------------------
B2: Vào thư mục muốn tạo, chọn cmd và nhập:

-  react-native start    (chạy cổng 8081 kết nối đến device)
